## Getting started

Install npm packages

```bash
npm install
```

Start dev server

```bash
npm run dev
```

- Access the default <http://localhost:5173>

### production level

```bash
## build
npm run build

## start
npm run preview
```

- Access the default <http://localhost:4173>
